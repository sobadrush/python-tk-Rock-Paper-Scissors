#  Rock Paper Scissors Game
這是一個使用 Tkinter 實現的簡單的猜拳遊戲。

## 安裝

1. 複製存儲庫：
  ```
  git clone https://github.com/sobadrush/python-tk-Rock-Paper-Scissors.git
  ```

2. 切換到項目目錄：
  ```
  cd rock-paper-scissors/
  ```

3. 安裝所需的依賴項：
  ```
  pip install -r requirements.txt
  ```

## 使用方法

要開始遊戲，執行以下命令：
```
python main.py
```

按照屏幕上的指示進行遊戲。
